변수란?

변수는 데이터를 저장하기 위한 이름표입니다. 파이썬에서는 변수를 선언할 때 별도의 키워드가 필요하지 않습니다.

# 변수 선언과 값 할당
message = "Hello, Python!"
count = 10
pi = 3.14

자료형 살펴보기

파이썬의 기본 자료형에는 문자열(str), 정수(int), 실수(float), 불린(bool) 등이 있습니다. type() 함수를 사용하면 변수의 자료형을 확인할 수 있습니다.

# 기본 자료형과 type() 함수
name = "Alice"
age = 25
height = 172.5
is_student = True

print(type(name))       # 
print(type(age))        # 
print(type(height))     # 
print(type(is_student)) #

리스트와 딕셔너리

여러 값을 하나의 변수에 담을 수 있는 자료형으로 리스트와 딕셔너리가 있습니다.

# 리스트와 딕셔너리 사용 예
fruits = ["apple", "banana", "cherry"]
person = {"name": "Alice", "age": 25}

print(fruits[0])       # apple
print(person["name"])  # Alice
